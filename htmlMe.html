HTML Tutorial

Introduction to HTML

HTML (HyperText Markup Language) is the standard language for creating web pages and web applications.
Example 1: Defines the document as HTML5.
<!DOCTYPE html>
Output: No visible output, but ensures standards mode.
Example 2: Root element of an HTML page.
<html lang="en"> ... </html>
Output: No visible output, but required for all HTML documents.
Example 3: The head contains meta info and the title for the browser tab.
<head><title>My Page</title></head>
Output: Browser tab shows: My Page
Example 4: The body contains all visible page content.
<body>Hello, world!</body>
Output: Hello, world!
Example 5: Sets the character encoding for the document.
<meta charset="UTF-8">
Output: No visible output, but ensures correct text display.
Example 6: Ensures proper scaling on mobile devices.
<meta name="viewport" content="width=device-width, initial-scale=1.0">
Output: No visible output, but enables responsive design.
Example 7: Links an external CSS file.
<link rel="stylesheet" href="style.css">
Output: No visible output, but applies styles.
Example 8: Links an external JavaScript file.
<script src="main.js"></script>
Output: No visible output, but runs scripts.
Example 9: HTML comments are ignored by browsers.
<!-- This is a comment -->
Output: No visible output.
Example 10: Sets the title in the browser tab.
<title>Welcome!</title>
Output: Browser tab shows: Welcome!
Example 11: Basic HTML document structure.
<html><head></head><body></body></html>
Output: No visible output unless content is added to body.
Example 12: Describes the page for search engines.
<meta name="description" content="A sample HTML page">
Output: No visible output, but helps SEO.
Example 13: Ensures compatibility with Internet Explorer.
<meta http-equiv="X-UA-Compatible" content="IE=edge">
Output: No visible output.
Example 14: Specifies the author of the page.
<meta name="author" content="Jane Doe">
Output: No visible output.
Example 15: Provides keywords for search engines.
<meta name="keywords" content="HTML, tutorial, web">
Output: No visible output.
Example 16: Sets the favicon for the page.
<link rel="icon" href="favicon.ico">
Output: Favicon appears in browser tab.
Example 17: Open Graph tag for social sharing.
<meta property="og:title" content="HTML Tutorial">
Output: No visible output, but improves sharing.
Example 18: Open Graph description for social sharing.
<meta property="og:description" content="Learn HTML">
Output: No visible output.
Example 19: Open Graph image for social sharing.
<meta property="og:image" content="image.png">
Output: No visible output.
Example 20: Tells search engines to index the page.
<meta name="robots" content="index, follow">
Output: No visible output.

Comments & Doctype

Comments are ignored by browsers. The doctype tells browsers which HTML version to use.
Example 1: HTML comment.
<!-- This is a comment -->
Output: No visible output.
Example 2: Doctype for HTML5.
<!DOCTYPE html>
Output: No visible output, but enables standards mode.
Example 3: Doctype for HTML 4.01 Strict.
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">
